Rules

Game Types:

There are two game types viz, 'Country to Flag' and 'Flag to Country'.

In the former you will be given a country name and you have to chose the correct flag of that country from the given choices.

While in the latter you will be shown a flag and you have to chose the correct country name from the given choices.

Game Modes:

There are two game modes viz, 'Arcade' and 'Quiz'.

In the former you will be given a series of questions. Your aim is to select the correct answer for all the questions. The game ends when you select an incorrect option.

While in the latter you will be given a set of ten questions and your aim is to select as many correct answers are possible. Unlike the previous mode, the game finishes only when you have attempted all the questions. At the end you will be given a score out of ten.
